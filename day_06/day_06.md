A condição while funciona da seguinte maneria:
~~~python
while something_is_true:
    #Do this
    #Then do this
    #Then do this
~~~

Funções em python podem ser criadas da seguinte maneira e são utilizadas para poupar o programador de ficar repetindo código:
~~~python
def my_function():
    #Do this
    #Then do this
    #Finally do this

#Calling the funcion
my_function()
~~~
